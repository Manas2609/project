# Forest Boundary Alarm Simulation

#.     python3 forest_alarm.py

This project simulates a forest area with animals moving inside it. If any animal touches the boundary, a siren will play.

## How to Run

1. Make sure Python is installed.
2. Install Pygame:
   ```
   pip install pygame
   ```
3. Add a siren sound file named `siren.wav` to this folder.
4. Run the program:
   ```
   python forest_alarm.py
   ```

## Files

- `forest_alarm.py` - Main simulation script
- `siren.wav` - Add your siren sound here (not included)

Enjoy!
